# Salary Management Tool - Planning

## User Persona
HR Manager managing 10k employees.

## Core Features

### Employee Management
- Add employee
- View employees
- Update employee
- Delete employee

### Salary Insights
- Min, Max, Avg salary per country
- Avg salary by job title in a country

### Additional Insights If Required
- Top 10 highest paid employees
- Salary distribution (optional)
- Department-wise average salary

## APIs

### Employee CRUD
- GET    /employees
- POST   /employees
- GET    /employees/:id
- PUT    /employees/:id
- DELETE /employees/:id
